Modeling Wine Quality with Machine Learning

Wine Quality

Project Overview

The goal of our project was to compare the performance of various machine learning classification models. The data used are quality ratings for red and white wine based on 11 physicochemical measurements in the wines. Red and white wine models are analyzed separately. There are 11 features for each wine type. Ten machine learning models were used.


Our datasets can be found here:


Red wine data.
White wine data.


The features used in this model:

  • fixed acidity
  • volatile acidity
  • citric acid
  • residual sugar
  • chlorides
  • free sulfur dioxide
  • total sulfur dioxide
  • density
  • pH
  • sulphates
  • alcohol

Target: recoded quality score (0-6 = Not Good (0) and 7+= Good (1))
